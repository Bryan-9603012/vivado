# VIVADO-LAB

A collection of Vivado FPGA laboratory projects.

## Repository Structure

- `0311/` : Vivado lab project
- `0318/` : Vivado lab project
- `0325/` : Vivado lab project
- `0408/` : Vivado lab project

Each lab folder usually contains:

- `src/` : Verilog source files
- `constr/` : constraint files (`.xdc`)
- `*.xpr` : Vivado project file
- `README.md` : lab description

## Toolchain

- Xilinx Vivado

## Notes

This repository is used to organize digital logic and FPGA practice projects.
Generated Vivado folders and temporary files are ignored by `.gitignore`.
